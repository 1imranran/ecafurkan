# 🔥 Furkan Yıldırım — Profesyonel Kombici Müşteri Avcısı Website

Kombi bakım, onarım, petek temizliği ve doğalgaz tesisatı alanında profesyonel müşteri avcısı (lead generation) tek sayfa web sitesi.

## 🚀 Hızlı Başlangıç

1. `index.html` dosyasını tarayıcıda açın
2. Site hazır! 🎉

## 📁 Dosya Yapısı

```
Kombici Kimlik/
├── index.html      # Ana sayfa
├── style.css       # Tasarım sistemi
├── script.js       # İnteraktif özellikler
├── images/
│   └── hero-bg.png # Hero arka plan görseli
└── README.md       # Bu dosya
```

## ✨ Özellikler

- **🎨 Premium Tasarım** — Glassmorphism, gradient animasyonlar, koyu tema
- **📱 Full Responsive** — Mobil, tablet, masaüstüne uyumlu
- **📞 Tek Tıkla Arama** — Telefon numarası ve WhatsApp doğrudan erişim
- **💬 WhatsApp Entegrasyonu** — Form doldurup WhatsApp üzerinden gönderim
- **🎯 SEO Optimizasyonlu** — Meta etiketleri, Open Graph, semantik HTML
- **⚡ Scroll Animasyonları** — IntersectionObserver ile performanslı animasyonlar
- **🔄 Otomatik Slider** — Müşteri yorumları otomatik geçiş
- **📊 Sayaç Animasyonu** — İstatistiklerde sayı animasyonu
- **📍 7/24 Floating Butonlar** — Sabit telefon ve WhatsApp butonları
